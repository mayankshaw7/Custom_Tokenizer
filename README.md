# Custom_Tokenizer
A lightweight, custom NLP Tokenizer built from scratch. It features modular implementations for Word, Character, and Sentence tokenization, designed for flexibility in text processing and model preparation. Ideal for projects requiring precise control over text segmentation without heavy external dependencies. 

## 🚀 Getting Started: Local Setup

Follow these steps to set up the project environment and run the application on your local machine.

### 1. Create a Virtual Environment
First, create a virtual environment to keep the project dependencies organized and isolated.
```bash
python -m venv venv
```

### 2. Activate the Environment
To start using the environment, you need to activate it. 

**On Windows:**
```bash
venv\Scripts\activate.bat
```
*(Once activated, you will see `(venv)` appear next to your command prompt line.)*

### 3. Run the Application
Open your project in **VS Code**, ensure your environment is active, and run the main application file:
```bash
python app.py
```

### 4. View the Project
Once the script is running, open your web browser and go to the following address to see the project live:
> **Host:** [http://127.0.0.1:5000](http://127.0.0.1:5000) (or the port specified in your terminal)

---

### 🛑 How to Stop
When you are finished working, you can exit the virtual environment by running:
```bash
venv\Scripts\deactivate.bat
```
